## ✨ Features

| Feature | Description |
|--------|-------------|
| ➕ **Add Tasks** | Instantly add new tasks with a single click |
| ✏️ **Edit Tasks** | Modify any task anytime without losing progress |
| ✅ **Complete Tasks** | Mark tasks as done with a satisfying checkmark |
| 🗑️ **Delete Tasks** | Remove tasks you no longer need |
| 💾 **Local Storage** | Your tasks are saved automatically — even after closing the browser |

## 💡 How to Use

1. **Type** your task in the input box
2. Press **Enter** or click **Add** button
3. Click the **✏️ icon** to edit a task
4. Click the **☑️ icon** to mark as complete
5. Click the **🗑️ icon** to delete a task
6. All tasks are **auto-saved** in your browser!
